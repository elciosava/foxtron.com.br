O que é HTML ?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele serve como a estrutura básica de um site, organizando o conteúdo em elementos como títulos, parágrafos, imagens, links, tabelas e listas. Os elementos HTML são marcados por tags, que indicam ao navegador como o conteúdo deve ser exibido. Por exemplo, a tag

define um parágrafo, enquanto insere uma imagem. Embora o HTML não seja uma linguagem de programação, ele é essencial para o desenvolvimento web, funcionando em conjunto com CSS (para o estilo) e JavaScript (para interatividade).
