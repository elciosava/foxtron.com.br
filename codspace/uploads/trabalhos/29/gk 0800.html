o que e html?

HTML, ou Linguagem de Marcação de Hipertexto, é a linguagem de programação padrão para criar a estrutura de páginas web, definindo o significado e a organização do conteúdo, como textos, imagens e links, para que navegadores possam exibi-los. É comparada ao esqueleto de um site, pois fornece a base para a exibição do conteúdo, que depois é estilizado com CSS e pode ter interatividade adicionada com JavaScript.
