oque é html? O HTML (HyperText Markup Language), ou Linguagem de Marcação de Hipertexto, é a base para a construção de páginas web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação usada para estruturar e organizar o conteúdo de um site.
