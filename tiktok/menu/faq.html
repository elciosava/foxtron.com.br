Perguntas Frequentes

O que é a tag <details>?

Ela serve para criar uma area que pode abrir e fechar, sem precisar de JavaScript

E a tag <summary>?

Ela é o titulo clicavel que aprace sempre visivel, funcionando como um botão para expandir o conteúdo.

Preciso de JavaScript para usar?

Não, basta usar essas duas tags que o navegador ja cuida do abrir e fechar.
