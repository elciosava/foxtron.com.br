Projeto de Criar um Forum interativo com opção de criar usuario e postagens

1. Modelagem do banco de dados mysql

opção de criar usuario: armazenar, id, nome, email e senha

postagens de topicos: armazenar conteudo que o usuario postou com id, conteudo, data e o id do usuario

comentarios de outros usuarios no topicos no caso respostas agradecimentos, duvidas. armazenar id do comentario, data, conteudo no caso o comentario, id do topico e id do usuario.

2. o fluxo de edição e exclução

1 - voce cria um formulario que ja vem pre preenchido com os dados atuais do banco de dados e clica em postar

2 - caso tenha erros o usuario pode editar o formulario e arrumar

3 - o usuario altera o texto e imagens e salva

4 - o php executa o update da atualização

Opção editar ou excluir conta de usuario

ter um pagina de configuração onde o usuario pode alterar nome de usuario, email, senha lembrando que o id sera fixo nao podendo alterar

opçao de exclução da conta onde usuario tera opção de excluir sua conta e o site perguntar se deseja mesmo executar aqui que se confirmar nao tera mais recuperação
