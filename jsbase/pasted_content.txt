Introdução
O que é a JSBase?

A JSBase é uma linguagem criada para ensinar
programação de forma simples.

Ela utiliza comandos em português
e posteriormente gera JavaScript.

Objetivo

• ensinar lógica

• preparar para JavaScript

• facilitar a aprendizagem

E agora vem a ideia que acho que vai fazer a JSBase se destacar
A documentação será gerada automaticamente.

Em vez de escrever cada página manualmente, vamos criar um sistema parecido com o do React ou Node.js:

Cada capítulo da documentação será um arquivo Markdown (.md).
O site terá um mecanismo que lê esses arquivos e monta automaticamente o menu lateral e o conteúdo.
Assim, adicionar um novo recurso à linguagem será tão simples quanto criar um novo arquivo Markdown.

Por exemplo:

docs/
├── introducao.md
├── primeiro-programa.md
├── variaveis.md
├── operadores.md
├── condicoes.md
├── repeticao.md
├── funcoes.md
└── biblioteca.md

Isso deixa a documentação organizada, fácil de manter e com aparência profissional, como grandes projetos open source.

Minha sugestão para a próxima etapa

Antes de adicionar mais recursos à linguagem, eu construiria a infraestrutura do projeto:

✅ Site oficial da JSBase.
✅ Documentação com menu lateral e busca.
✅ Playground integrado.
✅ Separar o compilador da interface (compiler.js), para que a mesma engine seja usada no playground, na documentação e futuramente em uma extensão para VS Code.

A partir daí, qualquer melhoria na linguagem beneficia automaticamente todo o ecossistema da JSBase. Acho que essa organização vai fazer o projeto crescer de forma muito mais sólida do que manter tudo em um único arquivo HTML.
