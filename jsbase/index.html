Documentação JSBase

Introdução

JSBase é uma linguagem de programação em português projetada para ensinar os conceitos fundamentais de programação, atuando como uma camada de abstração sobre JavaScript.

O objetivo é permitir que iniciantes aprendam lógica de programação sem a complexidade da sintaxe em inglês do JavaScript.

Características Principais

  • ✅ Sintaxe em Português
  • ✅ Transpilação em Tempo Real
  • ✅ Visualização do Código Gerado
  • ✅ Biblioteca Padrão Completa
  • ✅ Suporte a Funções
  • ✅ Destaque de Sintaxe

Instalação

JSBase é uma linguagem web-first. Para usar:

  1. Acesse o Playground
  2. Comece a digitar seu código
  3. Clique em "Executar" para ver os resultados

Primeiro Programa

Vamos criar nosso primeiro programa em JSBase:

mostrar "Olá Mundo!"

Resultado:

Olá Mundo!

Tipos de Dados

JSBase suporta três tipos de dados básicos:

Número

Representa valores numéricos inteiros ou decimais.

numero idade recebe 25
numero altura recebe 1.75

Texto

Representa sequências de caracteres. Deve estar entre aspas duplas.

texto nome recebe "João Silva"
texto mensagem recebe "Olá, Mundo!"

Booleano

Representa valores verdadeiro ou falso.

booleano ativo recebe verdadeiro
booleano inativo recebe falso

Variáveis

A declaração de variáveis segue o padrão:

<tipo> <nome> recebe <valor>

Exemplos

numero x recebe 10
texto nome recebe "Maria"
booleano aprovado recebe verdadeiro

Atribuição de Valores

Você pode atribuir novos valores a variáveis já declaradas:

numero contador recebe 0
contador recebe contador + 1

Operadores

Operadores Aritméticos

Operador Descrição Exemplo
+ Adição a + b
- Subtração a - b
* Multiplicação a * b
/ Divisão a / b
% Módulo (resto) a % b

Operadores de Comparação

Operador Descrição Exemplo
maior que Maior que a maior que b
menor que Menor que a menor que b
maior ou igual Maior ou igual a maior ou igual b
menor ou igual Menor ou igual a menor ou igual b
igual Igualdade a igual b
diferente Diferença a diferente b

Operadores Lógicos

Operador Descrição Exemplo
e E lógico (AND) condicao1 e condicao2
ou OU lógico (OR) condicao1 ou condicao2

Mostrar (Saída)

A função mostrar exibe valores no console de saída.

mostrar "Olá, Mundo!"
mostrar "O resultado é:", 42
mostrar "Múltiplos valores:", 1, 2, 3

Ler (Entrada)

A função ler solicita entrada do usuário através de um prompt.

texto nome recebe ler("Qual é o seu nome?")
mostrar "Bem-vindo,", nome

Condições: Se/Senão

Executa um bloco de código se uma condição for verdadeira.

se condicao
    // código executado se verdadeiro
senao
    // código executado se falso
fimse

Exemplo

numero nota recebe 8

se nota maior ou igual 7
    mostrar "Aprovado!"
senao
    mostrar "Reprovado!"
fimse

Laço: Enquanto

Repete um bloco de código enquanto uma condição for verdadeira.

enquanto condicao
    // código repetido
fimenquanto

Exemplo

numero i recebe 1

enquanto i menor ou igual 5
    mostrar "Número:", i
    i recebe i + 1
fimenquanto

Laço: Para

Repete um bloco de código um número específico de vezes.

para (inicializacao; condicao; incremento)
    // código repetido
fimpara

Exemplo

para (numero i recebe 1; i menor ou igual 10; i recebe i + 1)
    mostrar "Iteração:", i
fimpara

Funções

Definição de Função

funcao nomeDaFuncao()
    // corpo da função
fimfuncao

Função com Parâmetros

funcao soma(numero a, numero b)
    retornar a + b
fimfuncao

Chamada de Função

numero resultado recebe soma(10, 20)
mostrar resultado

Parâmetros e Retorno

Retorno de Valores

Use retornar ou devolver para retornar um valor da função.

funcao quadrado(numero x)
    retornar x * x
fimfuncao

numero resultado recebe quadrado(5)
mostrar "5² =", resultado

Função sem Retorno

funcao saudacao()
    mostrar "Bem-vindo ao JSBase!"
fimfuncao

saudacao()

Biblioteca Padrão

JSBase fornece uma biblioteca de funções úteis pré-implementadas:

Manipulação de Texto

  • tamanho(texto) - Retorna o número de caracteres
  • maiusculo(texto) - Converte para maiúsculas
  • minusculo(texto) - Converte para minúsculas

Números Aleatórios

  • aleatorio(min, max) - Gera número aleatório entre min e max

Data e Hora

  • hoje() - Retorna a data atual (DD/MM/YYYY)
  • hora() - Retorna a hora atual (HH:MM:SS)

Conversão de Tipos

  • inteiro(valor) - Converte para inteiro
  • decimal(valor) - Converte para decimal
  • texto(valor) - Converte para texto
  • booleano(valor) - Converte para booleano

Exemplos Completos

Calculadora Simples

funcao calcular(numero a, numero b, texto op)
    se op igual "+"
        retornar a + b
    senao
        se op igual "-"
            retornar a - b
        senao
            se op igual "*"
                retornar a * b
            senao
                retornar 0
            fimse
        fimse
    fimse
fimfuncao

mostrar "10 + 5 =", calcular(10, 5, "+")
mostrar "10 - 5 =", calcular(10, 5, "-")
mostrar "10 * 5 =", calcular(10, 5, "*")

Tabuada

funcao tabuada(numero n)
    para (numero i recebe 1; i menor ou igual 10; i recebe i + 1)
        mostrar n, "x", i, "=", n * i
    fimpara
fimfuncao

tabuada(7)

Verificar Número Par/Ímpar

funcao ehPar(numero n)
    se n % 2 igual 0
        retornar verdadeiro
    senao
        retornar falso
    fimse
fimfuncao

numero num recebe 42

se ehPar(num)
    mostrar num, "é par"
senao
    mostrar num, "é ímpar"
fimse

Tratamento de Erros

JSBase fornece mensagens de erro úteis quando algo dá errado:

  • Erro de Lógica: Blocos não fechados corretamente (fimse, fimenquanto, fimpara)
  • Erro de Execução: Problemas durante a execução do código JavaScript transpilado

Se um erro ocorrer, a mensagem será exibida em vermelho na saída do playground.
